# Змейка

Классическая игра «Змейка» в браузере.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 8080
```

Затем откройте http://localhost:8080

## Управление

- **Стрелки** или **WASD** — движение
- **Пробел** — начать игру / пауза / продолжить

## Правила

- Собирайте красные яблоки, чтобы расти и набирать очки (+10 за каждое)
- Не врезайтесь в стены и в свой хвост
- Рекорд сохраняется в браузере
